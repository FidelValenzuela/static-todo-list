# Static Todo List

A basic **Todo List** web app that helps keep track of daily tasks. Built with HTML and CSS.

## Features

- Dark theme with a simple, minimalistic design.
- Displays a list of predefined tasks.

## Technologies Used

- **HTML5**: Structure
- **CSS3**: Styling
